Startblock API Reference

Startblock alpha. All monetary values is to be sent and received as integers with the decimal numbers concatignated to the number e.g. 100 kr is to be represented as 10000.

Query url: http://146.185.140.4:3000/api/v1/

API Endpoint
http://localhost:3000/api/v1
Schemes: http
Version: 0.1.0

Paths

POST /company

Registers a new Company with the system and issues the shares.

NewCompany

Company data object for creation of new company in the system. The uid in the Company object is created by the and is ignored if included in the request.

Request Example
{
  "companyValue": "integer",
  "name": "string",
  "numShares": "integer(uint40)",
  "orgnr": "integer(uint32)"
}
200 OK

Successful response

400 Bad Request

Missing a required parameter.

Models

Company: object

uid: string

Global unique id of the company

name: string

The full company name e.g. Statoil ASA

orgnr: integer (uint32)

Organization number from Brønnøysundregistrene

numShares: integer (uint40)

The initial amount of shares issued e.g. 30000. Changes to the number of shares can be done by the admin of the company with functions like offering, split and splice

valueOfNumShares: integer

Total value

owners: object[]
